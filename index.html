Configurar sincronização ✦

Pra sua estante funcionar no celular e no computador com os mesmos dados, configure o Supabase. É gratuito e leva 3 minutos.

Passo a passo:
1. Cria conta grátis em supabase.com
2. Clica em "New project" (qualquer nome, senha e região)
3. Espera ~2 min até ficar pronto
4. Vai em SQL Editor e cola o SQL abaixo
5. Roda ▶ (Run)
6. Depois pega em Settings → API:
   • Project URL
   • anon public key
7. Cola aqui embaixo ✦
-- cole no SQL Editor e rode create table books ( id bigint generated by default as identity primary key, title text not null, author text not null, genre text not null, status text not null default 'pendente', start_date date, end_date date, rating int, favorite boolean default false, created_at timestamptz default now() ); alter table books enable row level security; create policy "acesso total" on books for all using (true) with check (true);
